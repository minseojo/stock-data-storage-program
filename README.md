# stock-data-storage-program

# 만든 이유
주식을 시작했는데, 주식을 시작하자마자 찜 해둔 주식 종류만 10개 넘었다. 근데 점점 늘어나는 찜 항목의 주식들을 내가 일일이 확인해서 메모해 둘 수는 없다.
그래서 크롤링을 통해 20230418.txt 형태로 하루마다 주가의 일일 데이터를 저장해 두려고 만들었다.
